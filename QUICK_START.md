## 🚀 STEP-BY-STEP DEPLOYMENT

### STEP 1: Clone Repository
```bash
git clone https://github.com/destinybenjob-netizen/hope4everychild-international.git
cd hope4everychild-international
```

### STEP 2: Setup Python Environment
```bash
python -m venv venv

# On Mac/Linux:
source venv/bin/activate

# On Windows:
venv\Scripts\activate
```

### STEP 3: Install Dependencies
```bash
pip install -r requirements.txt
```

### STEP 4: Configure .env File
```bash
# Copy example file
cp .env.example .env

# Edit with your credentials (use any text editor)
# nano .env  (Linux/Mac)
# code .env (VS Code)
```

### STEP 5: Initialize Database
```bash
python scripts/init_database.py
```

### STEP 6: Test Credentials
```bash
python scripts/test_credentials.py

# Should output:
# ✅ OpenAI: Connected
# ✅ Twilio: Connected
# ✅ Supabase: Connected
# ✅ PayPal: Connected
# ✅ Crypto Wallets: Connected
# ✅ All systems ready!
```

### STEP 7: Deploy to Railway (Free Hosting)

**Go to:** https://railway.app

1. Sign up with GitHub
2. Click "Create New Project"
3. Select "Deploy from GitHub repo"
4. Choose your repo
5. Add environment variables (from .env)
6. Deploy

**Railway will give you a live URL** - your system is now running 24/7

## 🆘 TROUBLESHOOTING

### Issue: "API key invalid"
**Solution:** Check .env file has correct key (no spaces)

### Issue: "Twilio SMS failed"
**Solution:** Verify WhatsApp sandbox setup at https://console.twilio.com

### Issue: "Database connection error"
**Solution:** Verify Supabase URL and API key in .env

### Issue: "No WhatsApp alerts"
**Solution:** 
1. Check phone number is verified in Twilio
2. Check YOUR_WHATSAPP_NUMBER in .env
3. Run: `python scripts/test_whatsapp.py`
